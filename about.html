About:

When someone does not know the meaning of a word or phrase, they usually just use Google to find its definition. However, they miss out on many key details of the word/phrase like synonyms, pronunciation, and concrete examples. This causes the user to not get a full understanding of the word or phrase. We are planning to create an HTML page where the user can enter a word/phrase and get a complete understanding by providing as much information as possible about the word/phrase. We hope that this website will help experienced English speakers learn more about the language, and also allow new learners to gain a thorough understanding of every aspect of the word. Whether it is a higher education student using our website to write more eloquent papers, or someone new to the language trying to gain a better understand of the English languages extensive verbage, we hope our website will satisfy a vast variety of stakeholder needs. Learning a new language, specifically English can be very daunting, so by incorporating multiple facets of different words such as synonyms, how to pronounce them, and phonetic spelling, we hope to help learners not feel bombarded by the many sources of information online.he information problem we are addressing is the issue of information overload for new language learners or new technology users. We hope to establish a “one-stop shop” for the English language. We are working with a dictionary API (https://dictionaryapi.dev/). The API will get us definitions, synonyms, pronunciation, and examples. The front end will be a simple input of the word through a text box, and when submitted it will return the data from the API backend such as definition, synonyms and antonyms.
